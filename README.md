A fully responsive web application that displays weather data fetched directly from NWS stations across the United States with interactive and stylized visuals. This data includes the current temperature, wind speed, wind direction, relative humidity, air pressure, and
detailed hourly forecasts for the week ahead. In addition, the app includes two interactive maps. One displays clouds, precipitation, wind, sea level pressure, temperature, and active tropical cyclones. The other displays the tracks and detailed information of all recorded
tropical cyclones worldwide since records began.
